What this does

For every post in D1, finds every static.wixstatic.com image URL in its body + cover, downloads it from Wix, uploads to your R2 bucket, and rewrites all references in the post HTML. Resumable; safe to re-run.

Heads-up: This will use R2 storage + Cloudflare egress. Estimated ~50 GB total for the full archive (10,000+ images). R2 free tier is 10 GB; overage is $0.015/GB/mo.
Smaller = safer. Each post averages ~7 images.
0 = start over. Use to resume after a stop.
