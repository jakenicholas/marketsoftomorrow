What this does

Enumerates every file in your Wix Media Manager (all folders) via the Wix Files API, then registers each one in the studio's media library.

Two passes, run in order:
1 · Index — registers all files pointing at their existing static.wixstatic.com URLs. Fast; makes the library browsable immediately. No storage used.
2 · Copy to R2 — downloads each file and stores it in your R2 bucket, then repoints the row at /media/<key> so you own it. ~17.6 GB ≈ $0.11–0.26/mo.
0 = auto (100 for Index, 20 for Copy). Wix caps this at 100. Lower if a batch times out.
