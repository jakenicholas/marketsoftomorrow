Setup

Before running the sync, the worker needs three Wix secrets and one admin token. Set them once via wrangler, then come back here.

  1. Generate Wix API key in Studio → manage.wix.com/account/api-keys with scope BLOG.READ-POSTS.
  2. From the worker repo: 
    wrangler secret put WIX_API_KEY
wrangler secret put WIX_SITE_ID
wrangler secret put WIX_ACCOUNT_ID  # if needed
wrangler secret put ADMIN_TOKEN     # if not already set
  3. cd worker && wrangler deploy
  4. Enter admin token below + click "Start sync".

Config

Same token as ADMIN_TOKEN secret. Stored locally only.

0 = start from newest. Use to resume after a crash.
