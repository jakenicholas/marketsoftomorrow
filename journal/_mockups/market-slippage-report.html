A note on how we build the forecast

The TMW Forecast is a statistical read on each project's predicted actual delivery, computed from the milestone cadence of comparable projects in our database — same city, same project type, same construction stage. Confidence dots reflect the sample size (3 = N≥10 comparables, 2 = 5-9, 1 = under 5). The window ± mo is the 68% interval, not a point estimate. Slippage is measured against the project's announced delivery at first publication, not the most recent stated date — a project that has already been re-announced three times still counts as slipping.
